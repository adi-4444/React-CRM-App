# This is CRM APP Create by React App

This project was made by React.

## Used Scripts in this project

In this Project Material-Table, Axios, React-Icons, Core-UI, Material-UI, React-Circular-ProgressBar are used.

### To run this project locally

clone this project by running in cmd `git clone https://github.com/adi-4444/React-CRM-App.git`.

go to cmd type `cd folder-name` and type `npm install` if got any error type `npm install --force`.

after installing packages type `npm start` then Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### To host run `npm run build` in cmd

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.
